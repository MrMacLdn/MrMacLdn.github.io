What is Machine Learning?


Machine learning is a branch of artificial intelligence "AI" and basically gives computers the ability to learn and discover algorithims without needing to explicitly be told what to do by a human programmer.

VS.

What is Traditional Programming?


Traditional programming, on the other hand is somewhat of the opposite. Traditional programming refers to a manually created program that uses inputted data and runs on a computer.

Examples of Machine Learning


Some real life examples of machine learning are;

Facial Recogniton


Facial recognition uses machine learning algorithims to analyze and process facial features allowing the system to learn patterns and characteristics.

Product Recommendation


The way that applications can recommend products to a customer accurately is through machine learning. The machine analyzes customer behavior, preferences, and purchase history if applicaple. The engine then can make a suggestion more likely to interest that particular customer.

Voice Recognition


Similar to facial recognition, machine learning is used to recognize voices and noises. The technology uses machine learning to process audio and convert it into words.

Examples of Traditional Programming


Some real world examples of traditional programming are:

Sales Analysis


Traditional programming uses set rules to determine an outcome, so it is used to read sales and apply those set rules and then output the end data.

Calculators


Something that nearly everyone is familiar with, a calculator. A calculator uses set algorithims to make calculations.

Programming Languages


Programming languages, used to create websites, or applications such as Java, C++, HTML are all examples of traditional programming.

Pros and Cons of Machine Learning


Pros


  • More Innovative
  • Improves Over Time
  • Automation Capabilities
  • More Efficient
  • Accurate Predictions

Cons


  • Data Dependant
  • Ethical Concerns
  • Short Term Cost
  • Job Displacement
  • Constant Repairs

Pros and Cons of Traditional Programming


Pros


  • Able to Create Complex Projects
  • Highly Customizable
  • More Researched
  • Full Control Over Programs
  • More Secure

Cons


  • Limited Automation
  • Less Flexible
  • Relies on Set Rules Instead of Changing Data
  • Difficult to Scale
  • More Suited for Simple Problems
